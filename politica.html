Política de Privacidade

Esta Política de Privacidade descreve Nossas políticas e procedimentos sobre a coleta, uso e divulgação de Suas informações quando Você utiliza o Serviço e informa sobre Seus direitos de privacidade e como a lei O protege.

Interpretação e Definições

Interpretação

As palavras cuja letra inicial é maiúscula têm significados definidos nas seguintes condições. As seguintes definições terão o mesmo significado, independentemente de aparecerem no singular ou no plural.

Definições

Para os fins desta Política de Privacidade:

  • Aplicativo refere-se ao Cronologia Arrowverse, o programa de software fornecido pela Empresa.
  • Empresa (referida como a "Empresa", "Nós", "Nos" ou "Nosso" neste Contrato) refere-se ao responsável pelo desenvolvimento do aplicativo Cronologia Arrowverse.
  • Dispositivo significa qualquer dispositivo que possa acessar o Serviço, como um celular ou um tablet digital.
  • Dados Pessoais: No contexto deste Aplicativo, não são coletados dados de identificação pessoal diretamente pelo desenvolvedor.
  • Serviço refere-se ao Aplicativo.
  • Dados de Uso referem-se aos dados coletados automaticamente, gerados pelo uso do Serviço (por exemplo, cliques em episódios e tempo de uso).

Coleta e Uso de Seus Dados Pessoais

Tipos de Dados Coletados

Dados Pessoais

Ao utilizar Nosso Serviço, o Aplicativo não solicita que Você forneça informações de identificação pessoal, como e-mail, nome ou número de telefone, para o funcionamento básico de checklist.

Dados de Uso

Os Dados de Uso podem ser coletados automaticamente ao usar o Serviço através de ferramentas de terceiros (como Firebase). Isso pode incluir o endereço IP do seu dispositivo, o modelo do aparelho, a versão do sistema operacional e dados de diagnóstico para melhoria do app.

Armazenamento Local

O progresso das suas séries (quais episódios foram marcados como assistidos) é armazenado localmente no seu dispositivo. Caso você desinstale o aplicativo ou limpe os dados do sistema, essas informações serão perdidas, pois não as mantemos em servidores externos.

Uso dos seus Dados Pessoais

A Empresa utiliza as informações coletadas para:

  • Para fornecer e manter nosso Serviço, garantindo que as imagens das temporadas sejam carregadas corretamente via Firebase Storage.
  • Para melhorar o Serviço: Análise de tendências de uso para saber quais funcionalidades são mais utilizadas.

Provedores de Serviços (Terceiros)

Podemos utilizar empresas terceirizadas para facilitar o Nosso Serviço:

  • Google Firebase: Usado para hospedar as imagens das séries e temporadas. O Google pode coletar dados de acordo com sua própria política de privacidade.

Segurança dos seus Dados Pessoais

A segurança dos seus Dados é importante para Nós. Embora utilizemos métodos comercialmente aceitáveis para proteger seus dados locais e o acesso às imagens, lembre-se de que nenhum método de transmissão pela Internet é 100% seguro.

Privacidade de Crianças

Nosso Serviço não coleta intencionalmente informações de identificação pessoal de menores de 13 anos. Como não solicitamos registro ou conta, não processamos dados de menores em nossos servidores.

Links para Outros Sites

Nosso Serviço contém links para outros sites (como bases de dados de séries). Se Você clicar em um link de terceiros, será direcionado ao site desse terceiro. Recomendamos fortemente que Você revise a Política de Privacidade de cada site que visitar.

Alterações nesta Política de Privacidade

Podemos atualizar Nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página para verificar quaisquer alterações. As alterações entram em vigor quando publicadas nesta página.
